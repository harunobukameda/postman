## Supported Language Versions

This SDK is compatible with the following versions: `TypeScript >= 4.8.4`

## Installation

To get started with the SDK, we recommend installing using `npm` or `yarn`:

```bash
npm install fromspec
```

or

```bash
yarn add fromspec
```

## Setting a Custom Timeout

You can set a custom timeout for the SDK's HTTP requests as follows:

```ts
const fromspec = new Fromspec({ timeout: 10000 });
```

# Sample Usage

Below is a comprehensive example demonstrating how to authenticate and call a simple endpoint:

```ts
import { Fromspec, NewRequestRequest } from 'fromspec';

(async () => {
  const fromspec = new Fromspec({});

  const newRequestRequest: NewRequestRequest = {
    userid: 'kameda',
  };

  const data = await fromspec.root.newRequest(newRequestRequest, {
    userid: 'string',
    time: '1484',
    accept: 'application/json',
  });

  console.log(data);
})();
```
